# Technologies Used

This project is built using the following technologies:

## Core Framework & Language
- **[Next.js](https://nextjs.org/)** (v14.2.3) - The React Framework for the Web.
- **[React](https://react.dev/)** (v18) - The library for web and native user interfaces.
- **[TypeScript](https://www.typescriptlang.org/)** - Strongly typed programming language that builds on JavaScript.

## Styling & UI
- **[Tailwind CSS](https://tailwindcss.com/)** - A utility-first CSS framework.
- **[Shadcn UI](https://ui.shadcn.com/)** - Re-usable components built using Radix UI and Tailwind CSS.
- **[Radix UI](https://www.radix-ui.com/)** - Primitives for building high-quality, accessible design systems.
- **[Framer Motion](https://www.framer.com/motion/)** - A production-ready motion library for React.
- **[Lucide React](https://lucide.dev/)** - Beautiful & consistent icon toolkit.
- **[React Icons](https://react-icons.github.io/react-icons/)** - Include popular icons in your React projects.
- **[Embla Carousel](https://www.embla-carousel.com/)** - A lightweight carousel library.
- **[Vaul](https://vaul.emilkowal.ski/)** - Drawer component for React.
- **[tsparticles](https://particles.js.org/)** - Lightweight TypeScript Particles library.

## Forms & Validation
- **[React Hook Form](https://react-hook-form.com/)** - Performant, flexible and extensible forms with easy-to-use validation.
- **[Zod](https://zod.dev/)** - TypeScript-first schema declaration and validation library.

## Backend & Database
- **[Mongoose](https://mongoosejs.com/)** - Elegant mongodb object modeling for node.js.
- **[NextAuth.js](https://next-auth.js.org/)** - Authentication for Next.js.
- **[Bcrypt](https://www.npmjs.com/package/bcrypt)** - A library to help you hash passwords.
- **[Axios](https://axios-http.com/)** - Promise based HTTP client for the browser and node.js.

## Development Tools
- **[ESLint](https://eslint.org/)** - Find and fix problems in your JavaScript code.
- **[PostCSS](https://postcss.org/)** - A tool for transforming CSS with JavaScript.
